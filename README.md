# ISupervision

## Description

ISupervision is a web app portal for students and lectures. Students can register for projects, bachelor theses and master theses. Lecturers can manage topics and supervise requests.

## Installation

The project runs with Docker Compose.

```bash
cd ISupervision
docker compose up --build
```

Before starting the application, configure the database settings in the hibernete configuration file.

## Usage

After starting the application, open it in your browser:

```text
http://localhost:8080
```

Students can browse and register for topics. Lecturers can manage topics and view registrations.
